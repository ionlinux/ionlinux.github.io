## Welcome to Ion Linux

Ion Linux is a modern, lightweight Linux distribution designed for performance and simplicity.

### System Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU       | x86_64  | x86_64 (4+ cores) |
| RAM       | 2 GB    | 8 GB        |
| Storage   | 20 GB   | 50 GB       |

### Quick Start

1. **Download** the latest ISO from the [download page](/download)
2. **Create** a bootable USB drive
3. **Boot** from the USB and follow the installer
4. **Enjoy** your new Ion Linux system
